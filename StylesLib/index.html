Библиотеки стилей

css/lib/borders.css

Библиотека-конструктор для создания различных границ объектов. Пространство имён: "border", например, "border-2px".

css/lib/buttons.css

Библиотека-конструктор для создания кнопок в стиле плоского дизайна. Пространство имён: "btn", например, "btn-primary".

css/lib/colors.css

Библиотека-конструктор для покраски блоков и текста. Пространоство имён: "color", например, "color-white-text"
